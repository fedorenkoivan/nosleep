Node is needed. Install it [here](https://nodejs.org/en).

To run application:
1. Clone this repository: `git clone https://github.com/fedorenkoivan/nosleep.git`
2. Verify that you in fact have node.js: `node --version`
3. Run `docker compose watch` in terminal

Access the UI: http://localhost:5173
Access the server: localhost: http://localhost:3000
